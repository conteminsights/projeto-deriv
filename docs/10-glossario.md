# 10 — Glossário

| Termo | Significado |
|-------|------------|
| **Deriv** | Corretora de trading online (antiga Binary.com) |
| **CALL** | Aposta que o preço vai subir |
| **PUT** | Aposta que o preço vai cair |
| **R_100** | Índice sintético da Deriv (volatilidade 100%) |
| **PAT** | Personal Access Token (token de API da Deriv) |
| **Proposal** | Cotação de um contrato antes da compra |
| **Buy** | Compra efetiva do contrato |
| **Tick** | Atualização de preço (mínimo incremento) |
| **Stake** | Valor apostado por contrato |
| **Payout** | Retorno potencial do contrato |
| **Martingale** | Progressão: dobra stake a cada loss |
| **Soros** | Estratégia de espelhamento MASTER/SLAVE |
| **Barreira** | Defesa que só permite operar após X losses |
| **Mini-meta** | Meta parcial de lucro que dispara reset |
| **Mxm entradas** | Máximo de entradas MASTER antes de reset |
| **Setup** | Configuração completa salva (páginas + banca) |
| **Nuitka** | Compilador Python → C++ |
| **PySide6** | Binding Python para Qt6 (GUI) |
| **Lightweight Charts** | Biblioteca de gráficos (TradingView) |
| **Zustand** | Biblioteca de estado global para React |
| **TailwindCSS** | Framework CSS utilitário |
